# 📋 Clipboard Manager

Простой и удобный менеджер буфера обмена, который автоматически сохраняет всё, что вы копируете.  
Все записи хранятся в истории. Достаточно нажать **Ctrl+Shift+Alt**, чтобы открыть окно и вставить любой ранее скопированный текст.

---

## ✨ Возможности

- 📋 Автоматическое сохранение всего, что копируется (Ctrl+C)
- 📌 Окно всегда поверх всех (не теряется за другими программами)
- 🎹 Глобальная горячая клавиша **Ctrl+Shift+Alt** (показать/скрыть окно)
- 🔕 Включение/отключение уведомлений (чекбокс в окне и в трее)
- 💾 Настройки сохраняются в `%APPDATA%\ClipboardManager\config.json`
- 🖥️ Иконка в системном трее
- 🎨 Тёмная тема (Catppuccin)

## 🚀 Установка и запуск (для разработчиков)

### 1. Клонировать репозиторий
```bash
git clone https://github.com/KORVONTEX/clipboard-manager.git
cd clipboard-manager
```
### 2. Установить зависимости
```bash
pip install -r requirements.txt
```
### 3. Запустить программу
```bash
python manager.py
```

---

## 🛠️ Используемые технологии

- **Python 3.13** — язык программирования
- **PyQt6** — графический интерфейс и системный трей
- **keyboard** — глобальные горячие клавиши
- **pyperclip** — работа с буфером обмена
- **PyInstaller** — сборка в .exe

---

## ⚠️ Примечания

- Уведомления включены по умолчанию (можно отключить чекбоксом)
- При первом запуске автоматически создаётся папка %APPDATA%\ClipboardManager
- История хранится до 50 записей
- Дубликаты не добавляются
- При нажатии на крестик окно скрывается в трей, а не закрывается
